# 贡献者公约行为准则

## 我们的承诺

作为项目维护者、贡献者和参与者，我们承诺为每一位参与者提供无骚扰的协作环境，无论年龄、体型、残障、种族、性别认同与表达、经验水平、教育程度、社会地位、国籍、外貌、宗教信仰或性取向如何。

## 我们的标准

有助于创造积极环境的行为包括：
- 使用友好和包容的语言。
- 尊重不同的观点和经验。
- 优雅地接受建设性批评。
- 关注对社区最有利的事情。
- 对其他社区成员表示同理心。

不可接受的行为包括：
- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或挑逗。
- 挑衅、侮辱/贬损评论，以及人身或政治攻击。
- 公开或私下骚扰。
- 未经明确许可发布他人的私人信息（如地址或电子邮件）。
- 在专业环境中可能被认为不适当的其他行为。

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，并暂时或永久禁止任何他们认为不适当、威胁、冒犯或有害的贡献者。

## 适用范围

本行为准则适用于项目空间内以及代表项目或社区时在公共空间中。代表项目的行为包括使用官方项目电子邮件地址、通过官方社交媒体账号发布内容，或在线上或线下活动中担任指定代表。

## 执行

如果发生滥用、骚扰或其他不可接受的行为，可以通过项目维护者（联系方式见 CONTRIBUTING.md）报告。所有投诉都将被审查和调查，并会给出必要且适当的回应。项目维护者有义务对事件的报告者保密。
